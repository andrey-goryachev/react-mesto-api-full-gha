# mesto-react
Клон проекта Mesto на чистом JavaScript, с использованием React, реализовано:
- адаптивная верстка
- всплывающие модальные окна
- при загрузке страницы, загружаются карточки с местами пользователя и профиль пользователя
- используется ООП

Технологии:
- HTML
- CSS
- JavaScript
- Grid
- Flex
- Webpack
- Babel
- React
